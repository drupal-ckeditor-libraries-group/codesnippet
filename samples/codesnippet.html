CKEditor Samples » Code Snippet

This editor is using the Code Snippet plugin, which brings beautiful code snippets. By default codesnippet plugin depends on built-in client-side syntax highlighting library highlight.js.

You can adjust the appearance of code snippets using codeSnippet_theme config variable (available themes).

Select template: 

CKEDITOR.replace( 'editor1', {
	extraPlugins: 'codesnippet',
	codeSnippet_theme: 'monokai_sublime'
} );

Please note that this plugin is not compatible with Internet Explorer 8.

Inline editor

The following sample shows Code Snippet running inside of inline CKEditor instance. 

CKEDITOR.inline( 'editable', {
	// Note: sourcedialog plugin is optional.
	extraPlugins: 'codesnippet,sourcedialog'
} );

Note: highlight.js themes must be loaded manually to be applied inside of inline editor instance, e.g. codeSnippet_theme will not work in that case. Include the stylesheet in <head> section of the page i.e.: 

<head>
	...
	<link href="path/to/highlight.js/styles/monokai_sublime.css" rel="stylesheet">
</head>

function isEmpty( object ) {
	for ( var i in object ) {
		if ( object.hasOwnProperty( i ) )
			return false;
	}
	return true;
}

Server–side highlighting and custom highlighting engines

Code Snippet GeSHi plugin is an extension of Code Snippet, which uses server-side highligter.

It also is possible to replace default highlighter with any library using Highlighter API and editor.plugins.codesnippet.setHighlighter().
